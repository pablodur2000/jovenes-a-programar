    Semana 02: Introducción al HTML

    En esta semana aprendimos cómo dar estructura a una página web usando HTML. HTML significa Hypertext Markup Language y es el lenguaje estándar usado para crear y diseñar páginas web. Para organizarlo se utilizan etiquetas, que a su vez estructuraran el contenido de las páginas web y definen elementos como encabezados, párrafos, enlaces, imágenes, formularios, etc. Las páginas web están pensadas para ser vistas mediante los programas llamados navegadores web, que interpretan estas páginas que en esencia son documentos de texto.

    Las etiquetas más comunes son:

    • La primera etiqueta !DOCTYPE es obligatoria e indica el tipo de documento (DOCTYPE) de la página. Existen varios tipos de documentos que corresponden a las diferentes versiones de HTML y XHTML. El tipo "html" es el que corresponde al HTML5.
    • head: Contiene metadatos y enlaces a recursos externos como hojas de estilo y scripts.
    • title: Especifica el título del documento que aparece en la pestaña del navegador.
    • body: Contiene el contenido visible de la página web.
    • h1 a h6: Definen encabezados, siendo h1 el más importante y h6 el menos importante.
    • p: Define un párrafo de texto.
    • a: Define un hipervínculo; el atributo `href` especifica la URL del destino del enlace.
    • img: Inserta una imagen; los atributos comunes son `src` para la fuente de la imagen y `alt` para el texto alternativo.
    • div: Define una división o sección en un documento HTML; se usa para agrupar elementos y aplicar estilos o scripts.

    Los elementos vacíos (los que no pueden contener texto) no tienen etiqueta de cierre. La etiqueta de apertura puede acabar en "/>". Las etiquetas vacías (void) de HTML son area, base, br, col, embed, hr, img, input, link, meta, param, source, track y wbr.

    Atributos

    Las etiquetas de apertura pueden contener uno o varios atributos. Los atributos especifican alguna característica de la etiqueta. Los atributos pueden a su vez tener valores (uno o varios valores, separados por espacios). El valor de cada atributo es asignado mediante el signo igual "=" (salvo en el caso de los atributos booleanos, que no requieren asignación). Cada atributo puede contener más de un valor, separados por un espacio, por lo que los valores no pueden contener espacios internos.

    Algunos de los atributos principales:

    • El atributo class permite definir clases de elementos. Cuando se define una clase de elemento, la regla se aplica únicamente a los elementos de ese tipo que tienen esa clase. Un elemento puede tener asignado varias clases.
    • El atributo id permite identificar de forma unívoca un elemento en un documento. Por ello, en una página web no puede haber dos elementos con el mismo valor en el atributo id.
    • El atributo style permite definir propiedades de estilo a un elemento determinado.

    Semana 03: Introducción al CSS

  • Qué es CSS?
      • Cascading Style Sheets CSS es un lenguaje que maneja el diseño y presentación de la páginas web se lo denomina hojas en estilo "en cascada" porque las reglas de estilo pueden ser aplicadas de forma jerárquica, permitiendo que estilos más específicos sobrescriban estilos más generales. CSS es fundamental para separar el contenido de una página HTML de su apariencia, facilitando la creación de un diseño visual coherente y consistente en un sitio web. La sintaxis básica de CSS se compone de selectores y declaraciones. Los selectores identifican los elementos HTML que se van a estilizar, mientras que las declaraciones definen las propiedades y valores que se aplicarán a esos elementos.
  • Como se implementa?
      • Se puede utilizar los estilos en linea directamente en el style del elemento, los estilos internos se implementan en el style del document del html, y por ultimo y el mas importante es el estilo externo el cual hay enlazarlo al html con la etiqueta link.
  • Selectores
    • Selector de tipo E: Si se escribe una etiqueta (E), las propiedades afectan a todos los elementos con etiqueta E.
    • Selector universal *:Si se escribe un asterisco (*), las propiedades afectan a todos los elementos de la página.
    • Selector de clase : E.e Si se escribe una etiqueta seguida de un punto y un nombre de clase (E.e), las propiedades afectan a todos los elementos con etiqueta E cuyo atributo class tenga el valor e.
    • Selector de identificador (id) E#e: Si se escribe una etiqueta seguida de una almohadilla y un nombre de id (E#e), las propiedades afectan al elemento con etiqueta E cuyo atributo id tenga el valor 'e'.
    • Selector de descendientes E F: Si se escriben dos etiquetas separadas por un espacio (E F) las propiedades afectan a los elementos con la segunda etiqueta (F) contenidos dentro de la primera etiqueta (E), aunque haya etiquetas intermedias.
    • Selector de hijos E > F: Si se escriben dos etiquetas seguidas separadas por un signo "mayor que" (E > F), las propiedades afectan a los elementos con la segunda etiqueta (F) contenidos dentro de la primera etiqueta (E), pero no lo selecciona si hay etiquetas intermedias entre ellos.
    • Selector de consecutivos E + F:Si se escriben etiquetas separadas por un signo "más" (E + F), las propiedades afectan únicamente a los elementos con la última etiqueta (F) que van justo después de un elemento con la etiqueta (E).
    • Selector de grupo E, F:Los selectores de grupo, son en realidad varios selectores 'unidos' para darles el mismo estilo a todos los elementos seleccionados.
  • Funcionalidades Basicas
    • Color: Define el color de texto, fondos y bordes utilizando palabras clave, RGB, HEX, HSL, etc.
      Ejemplo: color: red;, background-color: #f0f0f0;
    • Dimensiones: Define el ancho y alto de elementos.
      Ejemplo: width: 100px;, height: 200px;
    • Tipografía: Controla las propiedades de las fuentes, como tipo, tamaño, estilo y alineación.
      Ejemplo: font-family: Arial, sans-serif;, font-size: 16px;, text-align: center;
    • Espaciado: Gestiona el espacio alrededor de elementos mediante margin y padding.
      Ejemplo: margin: 10px;, padding: 20px;
    • Bordes: Define el estilo, color y grosor de los bordes.
      Ejemplo: border: 1px solid black;, border-radius: 10px;
    • Visualización (Display): Controla cómo se muestran los elementos en pantalla (block, inline, flex, etc.).
      Ejemplo: display: flex;, display: none;

    Semana 04: Introducción a JavaScript (DOM, HTTP y Fetch)

  • Como incluir un archivo JavaScript?
      • Para incluir un JS, debemos agregar la etiqueta 'script', dentro de esa etiqueta, se encuentra el atributo src. Con ese atributo, podemos determinar la ubicación del archivo JS a agregar. Se puede incluir en el 'head' o en el 'body'.
  • Estructura de codigo
      • En cuanto a la estructura de codigo, se recomienda tener una sentencia por linea de codigo, colocar el punto y coma al finalizar la sentencia, aunque puede ser omitido en algunos casos haciendo uso del salto de linea. Por ulktimo los comentarios pueden ser agregados en cualquier parte del codigo, los comentarios de multiples lineas pueden ser agregados con /* */, mientras que los comentarios de na linea pueden ser con //.
  • Funciones y variables
      • Las funciones, son utiles para organizar el código, hacerlo mas amigable y evitar código repetido. Dentro de la función, se detalla un codigo determinado y luego, mas adelante, se puede llamar a esa función cuando se necesite.
      Las funciones se declaran con la palabra clave 'function' seguido del nombre de la misma: function hola(parametro1){} - dentro de las llaves se detalla el código de la función. Luego, para llamar la funcion, se debe poner: hola(12);
      Dentro de los parentesis, se pueden especificar parametros que se pueden usar dentro de las función, estos van separados por una coma. En el ejemplo anterior, parametro1 = 12.
      Otro detalle a tener en cuenta, es que una función puede tener un return, esta palabra clave se usa cuando queremos que la función devuelva algún valor
  • DOM (Document object model)
      • Permite acceder y manipular los elementos de un HTML a través de nodos. DOM permite crear, eliminar un elemento, manipular una clase, etc. Sin necesidad de cambiar la página original.

      Tipos de nodos:
      Document: nodo raíz.
      Element: Repreenta cada una de las etiquetas HTML.
      Attrc: Representa cada uno de los atributos de las eti uetas de HTML.
      Text: nodo que contiene texto encerrado en una etiqueta HTML.

      Varias maneras de acceder a un determinado nodo:
      - getElementsByTagName("etiquetaHTML"): Devuelve una colección de todos los elementos con el nombre de la etiqueta especificada.
      - getElementsByName("nameHTML"): Devuelve una colección de todos los elementos con el atributo name especificado.
      - getElementById("id"): Devuelve el elemento que tiene el ID especificado.
      - querySelector("selector"): Devuelve el primer elemento que coincide con el selector especificado.
      - querySelectorAll("selector"): Devuelve una colección de todos los elementos que coinciden con el selector especificado.
  • JSON
      • Son cadenas de texto que se utilizan cuando se quieren trasmitir datos a traves de una red. Un objeto JSON puede ser almacenado en su propio archivo, que es un archivo de texto, con su estructura específica y con la extensión ".json". Dentro de un JSON, es posible incluir: cadenas, número s, arreglos, booleanos, y otros literales de objeto. Hay veces que se recibe una cadena JSON sin procesar, por lo que habra que procesarla con metodos de JavaScript. El metodo parse(): Acepta una cadena JSON como parámetro, y devuelve el objeto JavaScript correspondiente. El metodo: stringify(): Acepta un objeto como parámetro, y devuelve la forma de cadena JSON equivalente.
  • HTTP
      • (HyperText Transfer Protocol) está diseñado para permitir una comunicación entre clientes y servidores. Por ejemplo, alguien envia una request mediante una aplica oni solicitando informacion a un base de datos que es controlada mediante una API. Esta API va a dar una response al usuario con determinados datos. Los prrrogramadores deben utilizar esa response y mostrarsela al usuario en la interface de la aplciacion. Tenemos diferentes m todos: GET, POST, PUT, DELETE, PATCH. Los mas usados son el metodo GET, que sirve para obtener la informacion, el metodo POST q e funciona para insertar algun dato en la base de datos entre otras cosas y el metodo PUT para modificar algun dato de la BD.
  • FETCH
      • El metodo AJAX, permite pedir informacion a sevridores de forma asincronica, sin tener que refrescar la pagina. Para eso hoy en dia hay un estandar llamado FETCH. Esto se compone de la url y de las opciones que son paramentros opcionales al enviar la solicitud. Primero, la promesa, devuelta por fetch, se resuelve con un objeto del tipo RESPONSE tan pronto cómo el servidor (al que se le realizó el pedido) responda.
